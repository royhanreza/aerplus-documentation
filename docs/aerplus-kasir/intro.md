# Aerplus Kasir (Mobile App)

Selamat datang di dokumentasi teknis **Aerplus Kasir**.

Aerplus Kasir adalah aplikasi mobile Point of Sale (POS) yang dirancang untuk memudahkan transaksi penjualan, manajemen stok, dan operasional kasir pada perangkat iOS dan Android. Aplikasi ini terintegrasi dengan sistem backend Aerplus Admin untuk sinkronisasi data real-time.

## Tentang Aplikasi

Aerplus Kasir menyediakan fitur-fitur utama:

- **Transaksi Penjualan**: Proses checkout yang cepat dan mudah
- **Manajemen Pelanggan**: Pencatatan dan pencarian data pelanggan
- **Manajemen Produk**: Daftar produk dan stok real-time
- **Laporan Penjualan**: Riwayat transaksi dan laporan harian
- **Stock Opname**: Pengecekan dan pencatatan stok barang
- **Manajemen Pengeluaran**: Pencatatan pengeluaran toko
- **Transfer Barang**: Perpindahan stok antar outlet
- **Barcode Scanner**: Scan barcode untuk input produk cepat
- **Print Receipt**: Cetak struk pembayaran via Bluetooth printer
- **Push Notifications**: Notifikasi untuk order dan update penting

## Platform Support

- **iOS**: iOS 13.4 atau lebih baru
- **Android**: Android SDK 34 (Android 14) atau lebih baru
