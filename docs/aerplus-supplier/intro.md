# Aerplus Supplier (Mobile App)

Selamat datang di dokumentasi teknis **Aerplus Supplier**.

Aerplus Supplier adalah aplikasi mobile berbasis Flutter yang dirancang khusus untuk supplier/pemasok dalam ekosistem Aerplus. Aplikasi ini memudahkan supplier untuk mengelola dan memproses pengiriman produk (khususnya air minum) kepada outlet/toko dengan fitur tracking dan manajemen yang komprehensif.

## Tentang Aplikasi

Aerplus Supplier menyediakan fitur-fitur utama:

- **Water Delivery Management**: Manajemen pengiriman air ke outlet
- **Delivery Tracking**: Tracking status pengiriman real-time
- **Order Processing**: Proses order dari pending hingga completed
- **Delivery History**: Riwayat lengkap pengiriman
- **Profile Management**: Manajemen profil dan informasi supplier
- **Photo Upload**: Upload foto bukti pengiriman
- **Infinite Scroll**: List loading yang smooth dengan pagination

## Platform Support

- **Android**: Android 5.0 (API 21) atau lebih baru
- **iOS**: iOS 12.0 atau lebih baru
- **Web**: Support deployment ke web
- **Desktop**: Windows, macOS, Linux (optional)

## Teknologi

Built dengan:
- **Flutter**: Framework cross-platform untuk mobile development
- **Dart**: Modern programming language
- **GetX**: State management dan routing
- **Dio**: HTTP client untuk API communication
