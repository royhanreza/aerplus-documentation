# Toko Kita (Web App)

Selamat datang di dokumentasi teknis **Toko Kita**.

Toko Kita adalah aplikasi web berbasis Next.js yang dirancang sebagai platform belanja untuk customer/end user dalam ekosistem Aerplus. Aplikasi ini menyediakan pengalaman belanja yang praktis, cepat, dan hemat untuk kebutuhan harian dengan fitur ordering, payment, dan delivery tracking yang komprehensif.

## Tentang Aplikasi

Toko Kita menyediakan fitur-fitur utama:

- **Product Browsing**: Browse dan search produk dari outlet/toko
- **Shopping Cart**: Keranjang belanja dengan management yang mudah
- **Order Management**: Pembuatan order dan tracking status
- **Multiple Addresses**: Manajemen multiple delivery addresses
- **Payment Methods**: Berbagai metode pembayaran
- **Order History**: Riwayat pemesanan lengkap
- **Order Review**: Rating dan review untuk completed orders
- **Profile Management**: Manajemen profil customer
- **Maps Integration**: Mapbox untuk location selection
- **Real-time Updates**: Status order real-time

## Platform

- **Web Application**: Progressive Web App (PWA-ready)
- **Mobile Responsive**: Optimized untuk mobile browsers
- **Desktop Support**: Full desktop browser support

## Teknologi

Built dengan:
- **Next.js 15**: React framework dengan App Router
- **React 19 RC**: Latest React features
- **TypeScript**: Type-safe development
- **Tailwind CSS**: Utility-first CSS framework
- **DaisyUI**: Component library
- **Zustand**: Lightweight state management
- **React Query**: Data fetching dan caching
